Raw API

Raw API é um projeto focado na criação de uma API utilizando apenas recursos nativos da linguagem, sem a dependência de frameworks externos. O principal objetivo é entender e dominar os fundamentos por trás do funcionamento de uma API, lidando diretamente com aspectos como roteamento, tratamento de requisições e respostas, gerenciamento de erros e boas práticas de desenvolvimento. Esse projeto oferece uma visão aprofundada da base sobre a qual frameworks modernos são construídos.

REACTJS, NODEJS, SQL, Docker
GitHub repository

CRUD

CRUD é um projeto desenvolvido para aprofundar o conhecimento nas principais funcionalidades aplicadas no backend. O foco está na implementação das operações básicas de Create, Read, Update e Delete, fundamentais para a manipulação de dados em sistemas web. Durante o projeto, são explorados conceitos como criação de rotas, integração com bancos de dados, tratamento de requisições HTTP e aplicação de boas práticas no desenvolvimento de APIs. Este projeto é essencial para consolidar a base do desenvolvimento backend de forma prática e estruturada.

NODE, JAVASCRIPT, SQL, AXIOS, SEQUELIZE
